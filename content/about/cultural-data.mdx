## Overview

I worked in the 2,5 million EUR Horizon 2020 project, conducting research at the intersection of data science, cultural studies, and computational methods. My focuse is on understanding digital culture practices through quantitative and qualitative analysis.

## Research Areas

### Data Mining & Analysis
- Conducted large-scale data mining from social media platforms
- Developed data processing pipelines for cultural data analysis
- Applied statistical methods to identify patterns in digital behaviour

### Machine Learning
- Applied machine learning techniques to classify and analyse cultural content
- Built models for understanding social media engagement patterns
- Implemented natural language processing methods for text analysis

### Data Visualisation
- Created compelling data visualisations to communicate research findings
- Developed interactive visualisations for exploring cultural data
- Used Python visualisation libraries (Matplotlib, Seaborn) for publication-quality figures

### Social Media Research
- Investigated digital culture practices across multiple platforms
- Analysed user behaviour patterns and content consumption trends
- Published research findings in academic venues

## Methodologies

- Data mining and web scraping
- Machine learning and NLP
- Statistical analysis
- Network analysis
- Data visualisation
- Ethnographic methods
